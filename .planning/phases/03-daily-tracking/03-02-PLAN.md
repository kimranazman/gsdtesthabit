# Plan 03-02: Completion toggle with optimistic UI and server actions

## Goal
Implement one-tap completion toggling with immediate visual feedback using optimistic UI and server actions.

## Tasks
1. Create `src/lib/actions/completions.ts` with server actions:
   - `toggleCompletion(habitId, date)` - insert or delete completion record
   - `updateCompletionNotes(completionId, notes)` - update notes on existing completion
2. Create `HabitCheckItem` component with optimistic toggle
   - Uses `useOptimistic` or `useState` for immediate visual feedback
   - Calls server action in background
   - Shows loading state if action fails

## Files
- `src/lib/actions/completions.ts` - Server actions for completion CRUD
- `src/components/dashboard/habit-check-item.tsx` - Individual checklist item

## Verification
- Tapping a habit toggles completion with immediate visual feedback
- Toggle persists to database
- Optimistic UI reverts on error
